Solución Reto BCP

Bienvenido a la solucion del reto BCP. La solucion usa las sgtes tecnologias :

  • Angular and TypeScript for client-side code
    • Se crearon módulos para el routing
    • Se creó un pipe para la imagen aleatoria
    • Se persisten los filtros de busqueda entre la pantalla de listado y edición
    • La pantalla de edición de agencia solo valida el nombre
    • Se implementó un resolver para la naveción a la pantalla de edición
  • Bootstrap for layout and styling
